# SubTrack 測試指南

## 功能測試清單

### 1. 訂閱管理功能

#### 新增訂閱

- [ ] 開啟新增訂閱彈窗
- [ ] 填寫所有必填欄位
- [ ] 選擇圖示
- [ ] 選擇分類
- [ ] 設定價格和幣別
- [ ] 選擇扣款週期
- [ ] 設定下次扣款日期
- [ ] 提交表單
- [ ] 確認訂閱出現在列表中

#### 編輯訂閱

- [ ] 點擊訂閱卡片的編輯按鈕
- [ ] 修改訂閱資訊
- [ ] 儲存變更
- [ ] 確認變更已生效

#### 刪除訂閱

- [ ] 點擊訂閱卡片的刪除按鈕
- [ ] 確認刪除
- [ ] 確認訂閱已從列表中移除

#### 分類篩選

- [ ] 點擊「全部」標籤
- [ ] 點擊「娛樂」標籤
- [ ] 點擊「生產力」標籤
- [ ] 點擊「生活/其他」標籤
- [ ] 確認每個分類顯示正確的訂閱

- [ ] 確認兩種圖表都正確顯示

#### 統計明細

- [ ] 檢查分類明細列表
- [ ] 確認月費用計算正確
- [ ] 確認年費用計算正確

### 3. 設定功能

#### 帳號管理

- [ ] 註冊新帳號
- [ ] 登入現有帳號

- [ ] 確認登入狀態正確顯示

#### 雲端同步

- [ ] 上傳資料到雲端
- [ ] 從雲端下載資料
- [ ] 確認資料同步正確

#### 幣別設定

- [ ] 選擇不同的主要幣別
- [ ] 確認訂閱金額正確轉換
- [ ] 確認總覽卡片金額正確

#### 主題切換

- [ ] 切換到深色模式

- [ ] 確認所有元件顏色正確

### 4. 進階功能

#### 資料匯出

- [ ] 匯出 JSON 格式

#### 通知提醒 (Native only)
