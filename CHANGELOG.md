# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-16

### Added

- **訂閱管理**: 智慧追蹤所有訂閱服務，支援到期自動提醒。
- **預算分析**: 視覺化圖表（長條圖、圓餅圖）呈現支出趨勢，支援週/月/年統計。
- **多幣別支援**: 支援 TWD, USD, JPY 等 11 種貨幣，可自動轉換匯率。
- **雲端同步**: 整合 Firebase，支援跨裝置同步與資料備份。
- **離線模式**: 本地優先設計，無網路也可完整使用。
- **精美介面**: 支援深色/淺色模式，響應式設計適配多種裝置。
- **資料匯出**: 支援匯出為 JSON 或 CSV 格式。
- **開發工具**: 完整的 CI/CD 流程，整合 Husky, Lint-staged, ESLint, Prettier。

### Changed

- 初始化專案結構與核心功能實作。
- 優化效能與無障礙體驗 (a11y)。
